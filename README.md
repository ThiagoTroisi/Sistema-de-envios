# Sistema de Envíos

## Instalación del sistema

1. Ejecutar el instalador `Setup SistemaDeEnvios.exe`

2. Ingresar la instancia de SQL Server (ej: .\SQLEXPRESS o .)

3. Finalizar la instalación

---

## Configuración de base de datos

El sistema requiere una base de datos previa.

### Pasos:

1. Abrir SQL Server Management Studio
2. Conectarse a la instancia seleccionada en el instalador
3. Abrir el archivo `Query BD.sql` ubicado en la carpeta de instalación
4. Ejecutar el script completo
5. Verificar que se creó la base `SistemaDeEnvios`

---

## Ejecución

Luego de esto, ejecutar `SistemaDeEnviosGUI.exe`
